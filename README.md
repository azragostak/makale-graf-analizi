# Makale Graf Analizi

Bu proje, akademik makaleler arasındaki atıf ve referans ilişkilerini analiz etmek üzere geliştirilmiş bir Windows Forms uygulamasıdır.

## Özellikler

- Makale verilerini JSON formatından yükleme.
- Makaleler arasındaki atıf ilişkilerini graf (çizge) yapısı olarak kurma.
- **H-Core** ve **K-Core** analizleri ile önemli makale kümelerini belirleme.
- Görsel arayüz üzerinden makale ağlarını inceleme.

## Gereksinimler

- .NET Framework (veya .NET Core / 5+ projenin türüne göre)
- Visual Studio
